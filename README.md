## PA-System (Public Announcement System)
PA-System (Public Announcement) is a full-stack web developed during my internship 


# Project Purpose

- Practice full-stack web development

- Gain hands-on experience with REST APIs

# Features
- Speaker Group Management  
  Create, edit, delete, and display the status of speakers.
- Playlist Management  
  Create, edit, and delete playlists, as well as manage audio files within each playlist.
- Audio File Upload  
  Upload audio files create playlists.
- Live Announcement  
  Perform real-time (live) announcements through selected speakers.
- Playlist Playback  
  Play playlists on selected speakers with basic playback controls(play,pause,next,previous).
- Announcement History  
  View and track the history of announcements, including live broadcasts and playlist playback.
